Ejemplo: Confeccionar una pagina que muestre los nombres de 5 lenguajes de programacion separados por un guion.

                    <head>
                        <title>Primera practica</title>
                    </head>
                    <body>
                        <h2>Ejercicio 1: Estructura interna de una paginas</h2>

                        <p><h3>Resultado:</h3></p>
                        <p>- HTML - JavaScript - Python - C++ - Jason</p>
                    </body>
                </html>

Resultado:

- HTML - JavaScript - Python - C++ - Jason
